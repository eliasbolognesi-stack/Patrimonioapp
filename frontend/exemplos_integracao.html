🏛️ Exemplos de Integração - API Patrimônio

Exemplos práticos de como integrar o frontend com a API

ℹ️ Nota: Certifique-se de que a API está rodando em http://localhost:5000 antes de testar os exemplos abaixo.

1. Verificar Status da API

Verificar se a API está disponível
await PatrimonioAPI.checkHealth();

2. Listar Patrimônios

Obter lista de todos os patrimônios
const patrimonios = await PatrimonioAPI.getPatrimonios();
console.log(patrimonios);

3. Obter Patrimônio Específico

Buscar dados de um patrimônio pelo ID
const patrimonio = await PatrimonioAPI.getPatrimonio(1);
console.log(patrimonio);

4. Criar Novo Patrimônio

Adicionar um novo bem ao sistema
const novo = await PatrimonioAPI.createPatrimonio({
    descricao: "Notebook",
    categoria: "Informática",
    valor: 3500,
    localizacao: "Sala 101"
});

5. Dar Baixa em Patrimônio

Desativar um bem (dar baixa)
await PatrimonioAPI.darBaixa(1, "Motivo: Obsoleto");

6. Obter Relatórios

Gerar relatórios do sistema
// Patrimônios ativos
const ativos = await PatrimonioAPI.getRelatorioAtivos();

// Patrimônios inativos
const inativos = await PatrimonioAPI.getRelatorioInativos();

// Relatório geral
const geral = await PatrimonioAPI.getRelatorioGeral();

7. Validações e Formatações

Usar funções de validação e formatação
// Validar email
Validator.isValidEmail('teste@email.com');

// Formatar moeda
Formatter.formatCurrency(3500);

// Formatar data
Formatter.formatDate('2024-01-15');

// Verificar se campo está vazio
Validator.isEmpty('texto');

8. Armazenamento Local (SessionStorage)

Salvar e recuperar dados localmente
// Salvar dados
Storage.set('user', { name: 'João', email: 'joao@email.com' });

// Obter dados
const user = Storage.get('user');

// Remover dados
Storage.remove('user');

// Limpar tudo
Storage.clear();

📚 Resumo da API

GET - Listar

PatrimonioAPI.getPatrimonios()

GET - Específico

PatrimonioAPI.getPatrimonio(id)

POST - Criar

PatrimonioAPI.createPatrimonio(data)

PUT - Atualizar

PatrimonioAPI.updatePatrimonio(id, data)

DELETE - Deletar

PatrimonioAPI.deletePatrimonio(id)

POST - Baixa

PatrimonioAPI.darBaixa(id, motivo)

⚠️ Estes exemplos usam sessionStorage apenas para demonstração. Em produção, implemente autenticação segura com JWT ou cookies seguros.
